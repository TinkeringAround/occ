# One Click Checker ❤️

Test your App via popular Testing Sites in one single App.

![App-Screenshot](./designs/app.png 'App-Screenshot')

# Test your Sites with 12 popular Testing Tools

##### Server

- SSLLabs
- Security Headers
- Hardenize

##### Performance

- Lighthouse
- GTMetrix
- KeyCDN

##### SEO & Accessibility

- Seobility
- Varvy
- AChecker

##### Optimization

- Favicon-Checker
- W3 HTML Validation
- W3 CSS Validation
